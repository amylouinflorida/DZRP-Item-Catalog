# DZRP Item Catalog

## Mission

Help DayZRP staff find, understand, and use server items quickly.

## Core Goal

A staff member should be able to identify the correct item in under 10 seconds.

## Data Layers

### Official Data
Imported from DayZ/mod configs.

- Display name
- Classname
- Image
- Category
- Description
- Compatibility

### DZRP Server Data
Imported from Season files.

- Season
- Last updated
- Spawn status
- Trader status
- Item shop status
- Disabled/restricted status

### Staff Knowledge
Editable by staff.

- Tags
- Notes
- Common player descriptions
- Aliases

## Required Features

### Version 0.1
- Search items
- Show image
- Show display name
- Show classname
- Copy classname
- Show server status badge
- Back to search
- Staff tags display

### Later
- Add/delete staff tags
- Compatibility pages
- Item shop badges
- Trader badges
- Season data imports
- RCON spawning
