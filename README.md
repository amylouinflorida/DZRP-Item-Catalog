# DZRP Item Catalog

A staff-facing item search and catalog system for DayZRP.

## Planned Features

- Search by display name
- Search by classname
- Search by tags
- View item images
- Favorites
- Recently viewed items
- Future direct in-game spawning integration

## Initial Goals

- Build searchable item database
- Normalize inventory icons
- Support compensation requests
- Reduce dependence on VPP item browser

## Future Goals

- RCON integration
- Spawn items directly from catalog
- Teleport shortcuts
- Staff utility tools
